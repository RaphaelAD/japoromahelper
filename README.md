# japoromahelper

A light-weight ahk-script allowing you to write vowels with macrons by easy to reach key combinations, making it quicker to romanize those long japanese vowels.

Usage:
[Alt]+[A] ā
[Alt]+[Shift]+[A] Ā
[Alt]+[I] ī
[Alt]+[Shift]+[I] Ī
[Alt]+[U] ū
[Alt]+[Shift]+[U] Ū
[Alt]+[E] ē
[Alt]+[Shift]+[E] Ē
[Alt]+[O] ō
[Alt]+[Shift]+[U] Ō
